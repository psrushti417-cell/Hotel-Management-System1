
# Hotel Management System

## Project Overview

The Hotel Management System is a web-based application developed using Python and Django. It helps hotel staff efficiently manage hotel operations such as room management, customer registration, room booking, check-in/check-out, and reservation records. The system provides an easy-to-use interface for maintaining hotel information and improving operational efficiency.

## Features

* Customer Registration
* Room Management
* Room Booking
* Check-in and Check-out Management
* Reservation Management
* Admin Dashboard
* CRUD Operations
* Database Integration

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

2. Navigate to the project folder

```bash
cd Hotel_Management_System
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Apply migrations

```bash
python manage.py migrate
```

7. Run the development server

```bash
python manage.py runserver
```

8. Open in your browser

```text
http://127.0.0.1:8000/
```
