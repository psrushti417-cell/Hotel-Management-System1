# AI_USAGE.md

## AI Tool Used

* ChatGPT (OpenAI)

## Purpose of AI Usage

ChatGPT was used to assist with:

* Understanding GitHub repository setup.
* Creating the README.md file.
* Preparing project documentation.
* Explaining Git and GitHub commands.
* Reviewing documentation for clarity.

## Prompts Used

* "Create a professional README.md for my Hotel Management System."
* "How do I upload my Django project to GitHub?"
* "Help me prepare the required documentation for my project."
* "Explain the GitHub upload process step by step."

## Accepted Outputs

* README.md structure and content.
* GitHub upload guidance.
* Documentation format and project description.

## Modified / Rejected Outputs

* Updated the README to match the actual project features.
* Corrected installation steps where necessary.
* Removed any generic information that was not applicable.

## Mistakes Identified

* Some AI-generated descriptions included features that were not part of the project.
* These were manually reviewed and corrected before submission.

## Verification Steps

* Verified that all project files were uploaded successfully to GitHub.
* Reviewed the README.md after publishing.
* Confirmed that the project structure matched the repository.
* Checked that the documentation reflected the implemented project.
